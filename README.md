# build-aeo-content-brief

Build answer-ready SEO, AEO, and GEO content briefs from existing positioning, value proposition, product facts, proof, target questions, schema needs, and refresh requirements. Use when Codex needs to make content citeable by search and answer engines without inventing GTM messaging or Product decisions.

## Package Contents

- `SKILL.md` - the skill instructions.
- `references/source-map.md` - sanitized pattern map and extraction notes.
- `agents/openai.yaml` - optional agent configuration.
- `references/shared/` - shared reference files copied into this package for portability.

## Portability

This is a self-contained Codex skill package. Install or copy this folder as a standalone skill repository.
